# Лабораторные работы по параллельному программированию

## Lab 1 — MPI (Message Passing Interface)

В этой работе реализованы программы с использованием MPI для распределённых вычислений.

### Задачи и скрипты

#### 1. Вычисление числа pi (`pi_mpi`)
Параллельное вычисление числа pi методом интегрирования.

**Скрипт:** `run_pi_mpi.sh`
**Запуск:**
```bash
./run_pi_mpi.sh
```
Компилирует `pi_mpi.c` и запускает на 2 процессах.

---

#### 2. Измерение задержки MPI (`mpi_latency`)
Бенчмарк задержки и пропускной способности при передаче сообщений между двумя процессами.

**Скрипт:** `run_latency.sh`
**Запуск:**
```bash
./run_latency.sh
```
Компилирует `mpi_latency.c` и запускает на 2 процессах.

---

#### 3. Решение уравнения переноса
Реализовано два варианта: последовательный и параллельный (MPI).

**Последовательная версия (`transport_seq`)**
**Скрипт:** `run_transport_seq.sh`
**Запуск:**
```bash
./run_transport_seq.sh
```
Компилирует `transport_seq.c` и запускает с параметрами M=10000, K=10000.

**Параллельная версия (`transport_mpi`)**
**Скрипт:** `run_transport_mpi.sh`
**Запуск:**
```bash
./run_transport_mpi.sh
```
Компилирует `transport_mpi.c` и запускает на 2 процессах с параметрами M=10000, K=10000.

---

#### 4. Бенчмарк ускорения (`run_benchmark.sh`)
Сравнение производительности последовательной и параллельной версий уравнения переноса.

**Скрипт:** `run_benchmark.sh`
**Запуск:**
```bash
./run_benchmark.sh
```
Компилирует `transport_mpi.c` и запускает бенчмарк с разным числом процессов (1, 2, 4, 8, 16).

---

## Lab 2 — POSIX (процессы, потоки, каналы)

В этой работе реализованы программы с использованием POSIX API для многопоточных и многопроцессных вычислений.

### Задачи и скрипты

#### 1. Межпроцессное взаимодействие через pipe (`pipe`)
Измерение задержки при передаче данных между процессами через неименованный канал.

**Скрипт:** `run_pipe.sh`
**Запуск:**
```bash
./run_pipe.sh
```
Компилирует `pipe.c` и запускает программу.

---

#### 2. Межпоточное взаимодействие (`thread`)
Измерение задержки при синхронизации потоков с помощью мьютексов и условных переменных.

**Скрипт:** `run_thread.sh`
**Запуск:**
```bash
./run_thread.sh
```
Компилирует `thread.c` и запускает программу.

---

#### 3. Параллельная сортировка слиянием (`sort`)
Многопоточная реализация сортировки слиянием с бенчмарком производительности.

**Скрипт:** `run_sort.sh`
**Запуск:**
```bash
./run_sort.sh
```
Компилирует `sort.c`, запускает сортировку и сохраняет результаты в `results.csv`.

---

#### 4. Построение графика результатов (`plot_results.py`)
Визуализация результатов бенчмарка сортировки. Требуется python 3 с matplotlib.

**Скрипт:** `run_plot_res.sh`
**Запуск:**
```bash
./run_plot_res.sh
```
Запускает `plot_results.py`, который читает `results.csv` и строит график `sort_benchmark.png`.

---

## Требования

Для **Lab 1** требуется установленный MPI (например, OpenMPI или MPICH):
- Команды `mpic++` и `mpirun` должны быть доступны.

Для **Lab 2** требуются:
- Компилятор GCC с поддержкой pthread
- Python 3 с matplotlib для построения графиков (для `plot_results.py`)
